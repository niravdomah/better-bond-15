## Design Tokens & Style Guide â€” Summary

**Files written:** `generated-docs/specs/design-tokens.css` and `generated-docs/specs/design-tokens.md`

**Color decisions (MortgageMax, light mode only):**

| Token | Value | Notes |
|---|---|---|
| `--primary` | `#1A3A6E` navy | Exact MortgageMax brand navy |
| `--secondary` | `#0F766E` teal-700 | Interactive teal (passes AA white-on-teal) |
| `--accent` | `#CCFBF1` light teal | Hover/selected backgrounds |
| `--background` | `#F8FAFC` near-white | Reduces eye strain on data-dense screens |
| `--foreground` | `#0F172A` navy-tinted near-black | AAA contrast |
| `--card` | `#FFFFFF` pure white | Pops against near-white page |
| `--muted` | `#EFF3FB` light navy tint | Alt rows, skeleton |
| `--ring` | navy | Strong focus ring for keyboard users |
| `--destructive` | Shadcn default red | No brand conflict |

**Why teal-700 instead of teal-600 for buttons:** white-on-teal-600 only hits 3.74:1 (fails AA). Teal-700 hits 5.47:1. Teal-600 is reserved for charts, sidebar active indicators, and decorative highlights only.

**Other decisions:**
- **Typography:** Geist Sans system font, `text-sm` body, `tabular-nums` on all currency/count columns
- **Spacing:** `p-6` card padding, `gap-6` section gaps, `max-w-7xl` page container, 12-column grid for metric cards
- **Radius:** `0.5rem` (tighter than Shadcn default â€” suits financial data UI)
- **Chart palette:** 5 slots â€” navy â†’ teal-600 â†’ amber â†’ slate â†’ green
- **Top nav:** Navy bar, MortgageMax logo left, teal active underline, white inactive links
- **Currency format:** `R 1,234,567.89` via `Intl.NumberFormat('en-US')` (not en-ZA)
- **Toasts:** Sonner, 5s auto-dismiss
- **Reset Demo Data:** No confirmation dialog (per BR10 â€” fires immediately)
- **Dark mode block:** A `.dark` block exists in the CSS as a Shadcn `@theme inline` bridge, but the `.dark` class is never added to `<html>` â€” the app stays light mode per NFR3

**Accessibility:** All pairings verified at WCAG AA minimum; `disabled` attribute (not just `aria-disabled`) on inactive buttons; `aria-label` on row checkboxes.
